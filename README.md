STEPS TO RUN THE APP

1. Step 1: Clone the Repository
   git clone <PASTE_GITHUB_URL_HERE>
   Move into the newly created project folder:
   cd name-of-the-project

2. Step 2: Open the Project in VS Code
   code .

3. Step 3: Install Project Dependencies
   npm install

4. Step 4: Start the Expo Development Server
   npx expo start

5. Step 5: Open and Run the App
   Open the Expo Go app and tap "Scan QR Code"
